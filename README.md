# Contador-com-Mensagem-Personalizada

## Atividade Contador React

Atividade para praticar estados e componentes.

### Objetivo
Entender o uso de useState e a passagem de props.

### Estrutura
- App: Armazena o contador.
- Message: Exibe o texto.
- CounterButton: Executa o clique.

### Como funciona
O estado nasce no App e é enviado para os filhos via props. Ao clicar no botão, o valor atualiza automaticamente na tela.
